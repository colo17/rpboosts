# RPBoost Studio Landing (React + Vite + Tailwind)

## Requisitos
- Node.js 18+
- npm o pnpm o yarn

## Instalación

```bash
npm install
npm run dev
```

Abrí el navegador en el enlace que te aparezca (por defecto http://localhost:5173).

## Build para producción (Vercel / hosting)

```bash
npm run build
```

En Vercel elegí:
- Framework: **Vite**
- Build command: `npm run build`
- Output directory: `dist`
